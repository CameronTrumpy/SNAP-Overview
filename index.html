What is SNAP?

SNAP is an all-in one electronics toolkit that is purpose designed for students, by students. SNAP features four key electronics tools in a sleek, portable package. With an integrated Logic Analyzer, Oscilloscope, Waveform Generator, and Adjustable Power Supply, SNAP is designed to provide engineers all the tools they need to succeed when working on electronics.

Project Features

Measurement Module

The SNAP measurement module has two functions: a logic analyzer and an oscilloscope. The module utilizes a 10 bit ADC to read in analog inputs from a standard BNC cable and standard dupont wires for the digital input. It communicates data to the main module using SPI for offboard processing on either the web interface or Pulseview, which will display the oscilloscope and logic analyzer waveforms. The oscilloscope can accept any voltage between -20V to 20V, and the logic analyzer can accept 3V3 to 5V logic levels. The diagnostic red LED indicates when the module is powered. The Measurement Module can take measurements at up to 20MS/s on both inputs. Additionally, we support over 130 protocol decoders!

Output Module

The output module consists of a waveform generator and a variable power supply. The waveform generator is compatible with standard BNC connectors. It can generate Sine, Square, Triangle and Sawtooth waves from -5 V to 5V at up to 500kHz. The variable power supply exposes constant GND, 3.3V and 5V. It also exposes a configurable power supply, providing up to 140W, enough to power any small/medium electronics projects. All ports on the variable power supply are compatible with standard banana jacks.

Mainboard

The mainboard works as the brain of SNAP. You can power SNAP by connecting your laptop device to it with a USB-C (hopefully) cable – or through a 5V barrel jack to a wall outlet or power supply! You can then send commands to SNAP over USB using our custom GUI and it will relay them to the appropriate module. The mainboard will communicate with the modules over I2C for commands and SPI for high speed data transfer.
