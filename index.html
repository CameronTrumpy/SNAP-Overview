What is SNAP?

SNAP is an all-in one electronics toolkit that is purpose designed for students, by students. SNAP features four key electronics tools in a sleek, portable package. With an integrated Logic Analyzer, Oscilloscope, Waveform Generator, and Adjustable Power Supply, SNAP is designed to provide engineers all the tools they need to succeed when working on electronics.

Measurement Module

High‑Speed Logic Analyzer & Oscilloscope

  • 10‑bit ADC for analog signal capture via standard BNC inputs
  • Digital signal capture using standard Dupont connectors
  • Supports −20V to +20V analog input range
  • Accepts 3.3V to 5V logic levels
  • SPI communication to mainboard for high‑speed data transfer
  • View waveforms through PulseView or the SNAP web interface
  • Up to 20 MS/s sampling on both analog and digital channels
  • Over 130 supported protocol decoders

Output Module

Waveform Generator

  • Generates sine, square, triangle, and sawtooth signals
  • Output range from −5V to +5V
  • Frequencies up to 500 kHz
  • Standard BNC output connector

Adjustable Power Supply

  • Dedicated outputs for 3.3V, 5V, and ground
  • Configurable power rail capable of delivering up to 140W
  • Banana jack compatible connections

Mainboard

  • Acts as the central controller for all SNAP modules
  • Powered via USB‑C or standard 5V barrel jack
  • Custom GUI for sending commands and configuring modules
  • I²C communication for commands and system control
  • High‑speed SPI bus for transferring oscilloscope and analyzer data
